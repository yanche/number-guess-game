如何运行
1. 安装 nodejs: nodejs.org
2. 执行: node index.js 6 144

参数1为第N次猜出数字，参数2为最终猜出的数字
程序会给出所有的可能的组合

PS, 变量名随便起，参数没有做检查，没有错误处理

教授问第一个学生：你能猜出自己的数吗？回答：不能
问第二个，不能

再问第一个，不能

教授很满意的笑了。
